# Hands-On ML

- **Tokens and Embeddings:**
	Implementa un sistema de recomendación de canciones basado en Word2Vec. Utiliza un dataset de playlists para aprender representaciones vectoriales (embeddings) de canciones, permitiendo identificar temas similares mediante la proximidad de sus vectores en el espacio latente.

- **Text Classification:**
	Compara diversos métodos de clasificación de sentimientos (RoBERTa, FLAN-T5 y GPT) aplicados al dataset de Rotten Tomatoes. Explora desde el uso de embeddings con regresión logística hasta técnicas de zero-shot classification y modelos generativos.
